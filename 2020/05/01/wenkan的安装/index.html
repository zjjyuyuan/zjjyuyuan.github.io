01-wenkan的安装

公司内网里面没法装，为了方便先在自己家里装好然后虚拟机传到公司，升级自己的Wekan面板

查找文档（Github原文）

^1

  1. 与Windows Node + MongoDB捆绑

    • 它具有最高的性能和最低的RAM使用率，因为没有像Docker，Windows Linux子系统之类的虚拟化技术。Wekan在Windows文件系统上直接使用Windows本机版本的Node.js和MongoDB运行。如果您在Wekan中有重要数据，请进行备份。

  2. 为Windows安装最新的Node.js v12.xx https://nodejs.org/dist/v12.16.1/

  3. 为Windows安装MongoDB 4.x https://www.mongodb.com/download-center/community

  4. 从https://releases.wekan.team下载和最新的Wekan软件包wekan-3.xx.zip

  5. 解压缩wekan-3.xx.zip，它具有目录名称 bundle

（这不是必需的）像在bundle wiki页面上一样，对于bundle也同样如此：

cd bundle/programs/server
npm install
npm install node-gyp node-pre-gyp fibers
cd ..\..\..

  1. 将start-wekan.bat下载到您的bundle目录。根据您的ROOT_URL等设置对其进行编辑。

  2. 启动Wekan：

  3. cd bundle

  4. start-wekan.bat

  5. 添加用户。
